### Key Projects 🚀

#### 🎮 KODA — Scholastic Esports LMS
*Built with React 19, TypeScript, & Firebase*
An infrastructure layer for K–12 esports programs, currently targeting **700 Title I schools** in NYC. Secured **funding** and a **grant** from the NYC Mayor's Office.
- **📚 Stack:** React 19, TypeScript, Firebase (Firestore/Auth/Hosting), Google Gemini 2.0 Flash
- **🎀 Features:** Role-based access (Student/Coach/Recruiter), Real-time bracket engine, and **KODA Coach** (AI chatbot)
- **🔒 Compliance:** Built to FERPA and COPPA standards for safe student-recruiter interaction

#### 📱 Ledes — Category-Selective Microblogging
*Built with React Native & Node.js*
A social mobile app featuring a novel "lede" system that allows users to subscribe to specific content buckets (e.g., Tech vs. Sports) from creators.
- **📚 Stack:** React Native 0.81 (Expo), TypeScript 5.9, Node.js (Express 5), PostgreSQL
- **🎀 Features:** Category-selective following, gesture-driven UI (Reanimated), and JWT-based silent auth
- **🛠️ Engineering:** Implemented comprehensive validation, optimistic UI updates, and full-text search
